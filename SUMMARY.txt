HOW TO USE
================================================================================

1. OPEN THE APP
   → Navigate to: http://localhost:8000

2. SWITCH VERSIONS
   → Use the "Version" dropdown in the header
   → V1 = Core Roadmap (17 items)
   → V2 = Extended Roadmap (40 items)

3. EXPLORE VIEWS
   → Timeline: See items by quarter
   → Grid: Browse in card layout
   → List: View compact list

4. SEARCH & FILTER
   → Type in search box
   → Select category filter
   → Select status filter

5. VIEW DETAILS
   → Click any item for full details
   → See key features and impact
   → Close with X or Esc key

6. EXPORT DATA
   → Click Export button
   → Downloads JSON of current view

CUSTOMIZATION
================================================================================

To Update Roadmap Data:
  1. Open: ~/agentforce-roadmap/data.js
  2. Edit roadmapDataV1 array (IDs 1-17)
  3. Edit V2-specific items (IDs 18-40)
  4. Save and refresh browser

  1. Open: ~/agentforce-roadmap/styles.css
  2. Edit CSS variables in :root
  3. Save and refresh browser

To Add New Items:
  1. Follow data structure in data.js
  2. Use next available ID number
  3. Include all required fields
  4. Add to appropriate version array
